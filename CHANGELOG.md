## 1.0.0

* Initial release
* Video compression with quality parameter (0-100)
* Native implementations:
  * Android: Media3 Transformer API
  * iOS: AVFoundation (AVAssetWriter)
  * macOS: AVFoundation (AVAssetWriter)
  * Windows: Media Foundation API
* Preset configurations: WhatsApp, Social, Light, Maximum
* Progress tracking with real-time callbacks
* Compression cancellation support
* Video metadata extraction (MediaInfo)
* Thumbnail generation
* Cache management
